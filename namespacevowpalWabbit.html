Detailed Description

A JNI layer for submitting examples to VW and getting predictions back.

It should be noted that at this time VW is NOT thread safe, and therefore neither is the JNI layer. It should be noted that this was originally written with a bulk interface that was later removed because of benchmarking data found here.

NOTE: When finished using VW predictors, close MUST be called in order to free up the memory on the C side.
