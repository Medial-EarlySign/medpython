void Incident_Stats::read_from_text_file ( const string &  text_file)

Reading the file.

the file format is:

  • a line with AGE_BIN[TAB]{NUMBER}
    AGE_BIN is keyword, and {NUMBER} is the age bin value numeric
  • a line with AGE_MIN[TAB]{NUMBER}
    AGE_MIN is keyword, and {NUMBER} is the age minimal value
  • a line with AGE_MAX[TAB]{NUMBER}
    AGE_MAX is keyword, and {NUMBER} is the age maximal value
  • a line with OUTCOME_VALUE[TAB]{NUMBER}
    OUTCOME_VALUE is keyword, and {NUMBER} is a possible outcome value. binary bootstrap will contain 2 lines with 0 and 1 values
  • a line with STATS_ROW[TAB]{MALE|FEMALE}[TAB]{AGE_NUMBER}[TAB]{OUTCOME_VALUE}[TAB]{NUMBER_COUNT}
    STATS_ROW is keyword, and than you provide either "MALE" or "FEMALE", TAB the age TAB the outcome value (in binary 0 for control, 1 for case) TAB the count. the incidence will calulate the incidence rate in each group.
