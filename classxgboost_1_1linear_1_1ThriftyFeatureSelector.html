Detailed Description

Thrifty, approximately-greedy feature selector.

Note
Prior to cyclic updates, reorders features in descending magnitude of their univariate weight changes. This operation is multithreaded and is a linear complexity approximation of the quadratic greedy selection.
It allows restricting the selection to top_k features per group with the largest magnitude of univariate weight change, by passing the top_k value through the param argument of Setup().
