)

Save model into R's raw array.

Parameters
handlehandle
json_configJSON encoded string storing parameters for the function. Following keys are expected in the JSON document:

"format": str

  • json: Output booster will be encoded as JSON.
  • ubj: Output booster will be encoded as Univeral binary JSON.
  • deprecated: Output booster will be encoded as old custom binary format. Do now use this format except for compatibility reasons.
