Detailed Description

Vowpal Wabbit wrapper for anonymous classes.

Type used for serialization doesn't need to be known at compile time, but it's checked at runtime.

For each call to Learn there is additional overhead as the type is looked up in a dictionary compared to VowpalWabbit<T>.
