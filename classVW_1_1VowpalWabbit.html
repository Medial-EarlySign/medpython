Detailed Description

Simple string example based wrapper for vowpal wabbit.

VW wrapper for multiline ingest.

VW wrapper supporting data ingest using declarative serializer infrastructure.

If possible use VowpalWabbit{T} types as this wrapper suffers from marshalling performance wise.

TActionDependentFeatureThe user type for each action dependent feature.
