Detailed Description

template<class Ctx>
class MultiWorldTesting::SingleAction::IPolicy< Ctx >

Exposes a method to choose an action given a generic context.

IPolicy objects are passed to (and invoked by) exploration algorithms to specify the default policy behavior.
