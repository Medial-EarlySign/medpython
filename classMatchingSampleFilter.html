Detailed Description

Matching filter

  • match cases and controls according to a set of matching strata
  • matching creiterion can be
    • value of signal within window
    • age
    • sample time

  • Each sample is assigned to a bin according to the vector of strata. The optimal case/control sampling ratio is then found, when each control that is removed costs 1 point and each case costs eventToControlPriceRatio points. Once ratio is decided, sampling is perfored randomly within each bin.
